Target Generator

Description

This simple program allows the user to manipulate shapes in order to create unique patterns and styles. This was coded using a combination of Javascript, HTML, & CSS.

Parameters

  • Rectangles: This controls the amount of rectangles that will appear on the screen.
  • Distance: This control how far apart each rectangle will be from the center of the original.
  • Stroke Weight: This controls the thickness of each rectangle.
  • Background: This controls the color of the background.
  • Shape Color: This controls the color of each rectangle.
